Hexo GitHub搭建个人静态博客

搭建过程大概需要这么几个步骤：

详细资料：http://luhongquan.cn/posts/4f5b69ce

1.搭建环境的准备工作（主要是node.js和git环境，以及GitHub账户的设置）

1、安装npm 
2、本地Git环境搭建
3、GitHub账号注册
4、配置github，创建New respository仓库，选择public
配置setting的github pages ，有个按钮 Choose a theme 选择一个主题后保存基本上初页就完成了。可以访问如：https://raochuan.github.io/

2.安装并初始化Hexo
为什么要装hexo,使博客有漂亮的皮肤，使用本地环境编译和快速发布Markdown编辑后的文档

1、新建文件夹 mkdir ‘hexo-page’
2、安装hexo,npm install hexo-cli -g 请科学上网或者改镜像
3、保存项目 npm install hexo --save
4、检测hexo安装 hexo -v
5、初始化本地脚手架 hexo -init，npm install更新框架
6、编译和本地调试 hexo g /hexo -s
本地可以调试和浏览

3.将Hexo和GitHub Page联系起来
主要是关联SSH，配置SSH keys

1、非第一次使用git生成ssh,用ssh-keygen -t rsa -C "邮件地址@youremail.com"在命令行执行
2、Mac前往文件夹 ~/.ssh/找到id_rsa.pub文本打开
3、赋值keys到github的setting里面的ssh and GPG keys
4、检测是否SSH配置成功 ssh -T git@github.com，未成功需要配置git config全局的邮箱账号
5、找到本地项目的_config.yml配置如下

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
    type: git
    repo: git@github.com:raochuan/raochuan.github.io.git
    branch: master

4.如何发布新文章

1、hexo new post "article title"创建新的.MD文件
2、在生成和部署文章前安装插件npm install hexo-deployer-git --save 避免连接不上github
3、生成:hexo g / 部署:hexo d
